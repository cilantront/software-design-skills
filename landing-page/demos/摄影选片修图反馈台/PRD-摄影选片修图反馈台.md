# 1. 产品概述

## 1.1 产品定位

摄影选片修图反馈台是一款面向中小摄影工作室的垂直行业 SaaS 协作工具。产品聚焦摄影交付链路中最碎片化的环节——"选片 → 修图反馈 → 交付"，通过结构化的在线流程替代微信反复传图、网盘反复分享的混乱协作模式。

**核心价值主张**：
- 对客户：免安装、打开即用，手机端流畅选片和反馈
- 对工作室：订单进度一目了然，反馈不再遗漏，交付周期可量化缩短
- 对修图师：任务集中管理，修改意见精确到照片和版本，告别"改错版本"

## 1.2 目标用户

| 用户类型 | 典型画像 | 核心诉求 |
| --- | --- | --- |
| 工作室管理员/老板 | 3-8 人团队，月 20-50 单 | 掌控全局进度、减少客户投诉 |
| 独立摄影师 | 1-2 人，月 8-15 单 | 简化交付流程、专注拍摄 |
| 门店客服 | 前台/在线客服 | 高效跟进多订单状态 |
| 修图师 | 专职/兼职 | 清晰的任务列表、精确的修改意见 |
| C端客户 | 婚纱/亲子/证件照/商业客户 | 方便快捷的选片和反馈体验 |

## 1.3 产品边界

**MVP范围内**：
- 工作室管理后台（Web端，≥1280px）
- 客户选片 H5 页面（移动端优先）
- 订单三态看板
- 云端图片存储与管理
- 基础通知体系

**MVP范围外**（后续版本）：
- 预约排期系统
- 在线支付（套餐费）
- 相册模板设计工具
- 社交分享功能
- 视频编辑和选片

## 1.4 商业模式

| 套餐 | 价格 | 核心限制 |
| --- | --- | --- |
| 免费版 | ¥0/月 | 每月 10 个订单、5GB 存储、平台水印 |
| 工作室版 | ¥69/月 | 不限订单数、不限存储、自定义品牌水印、多修图师协作、版本记录 |

# 3. 页面详细说明

## 3.1 工作室管理后台 — 首页/看板总览

### 3.1.1 页面目标

为工作室管理员和客服提供一站式订单进度总览，快速了解各阶段待处理事项。

### 3.1.2 页面布局

| 区域 | 内容 | 交互 |
| --- | --- | --- |
| 顶部导航栏 | 工作室名称/LOGO、全局搜索、通知铃铛、用户头像菜单 | 通知铃铛点击展开下拉列表；用户头像下拉含"个人设置""退出登录" |
| 统计卡片区 | 四张数据卡片：今日新增订单、待修图、待客户确认、待交付 | 点击卡片跳转至对应看板列 |
| 三态看板区 | 三列：待修图 \| 待客户确认 \| 待交付，每列展示订单卡片 | 卡片可拖拽（仅改分配人），点击跳转订单详情 |
| 超时提醒区 | 超过约定交付日期未推进的订单列表 | 点击跳转订单详情 |

### 3.1.3 订单卡片信息

| 字段 | 说明 |
| --- | --- |
| 订单编号 | 系统自动生成，如 "ORD-20260626-001" |
| 客户姓名 | 脱敏显示（如 "张*"） |
| 拍摄类型 | 标签形式：婚纱 / 亲子 / 证件照 / 商业 |
| 当前状态 | 彩色状态标签 |
| 负责人 | 修图师头像+名称 |
| 更新时间 | 最后操作时间，如 "2小时前" |
| 精修进度 | "3/10 张" 进度条 |

### 3.1.4 交互规则

- 看板数据每 30 秒自动刷新一次
- 超时订单（超过套餐约定天数）以红色边框高亮
- 空列显示"暂无订单"占位图

## 3.2 订单管理 — 订单列表页

### 3.2.1 页面目标

展示工作室所有订单，支持按状态、时间、客户名等维度快速筛选。

### 3.2.2 页面布局

| 区域 | 内容 | 交互 |
| --- | --- | --- |
| 页面标题区 | "订单管理" + "创建订单"按钮 | 点击按钮跳转创建订单页 |

| 订单表格 | 订单编号、客户姓名、拍摄类型、精修张数、当前状态、创建时间、操作 | 每行操作：查看详情、复制选片链接 |

### 3.2.3 状态标签配色

| 状态 | 配色 |
| --- | --- |
| 待上传原片 | 灰色 |
| 待选片 | 蓝色 |

| 待客户确认 | 紫色 |

## 3.3 订单管理 — 创建订单页

### 3.3.1 页面目标

客服或管理员快速创建新拍摄订单。

### 3.3.2 表单字段

| 字段 | 类型 | 必填 | 校验规则 |
| --- | --- | --- | --- |
| 客户姓名 | 文本输入 | 是 | 2-20 字符 |
| 客户手机号 | 文本输入 | 是 | 11 位手机号格式校验 |
| 拍摄类型 | 单选 | 是 | 婚纱 / 亲子 / 证件照 / 商业 |
| 套餐名称 | 文本输入 | 否 | 如"婚纱照 ¥3999 套餐" |
| 精修张数 | 数字输入 | 是 | 1-999 正整数 |
| 约定交付日期 | 日期选择 | 否 | 不早于今天 |
| 备注 | 多行文本 | 否 | 500 字符以内 |

### 3.3.3 交互规则

- 手机号格式不正确时实时提示
- 提交成功后跳转至订单详情页，toast 提示"订单创建成功"
- 创建后订单状态为"待上传原片"
- 系统自动发送选片链接短信给客户（P1，MVP后）

| Tab 切换区 | "原片管理" / "选片结果" / "修图协作" / "交付管理" 四个 Tab |

#### Tab 1：原片管理

| 功能 | 描述 |

| 原片列表 | 网格布局展示已上传原片，缩略图+文件名+上传时间 |
| 分组标记 | 可按场景/服装对原片进行分组（P1） |
| 删除原片 | 支持删除已上传的原片（需二次确认） |

#### Tab 2：选片结果

| 功能 | 描述 |
| --- | --- |
| 选片概览 | "客户已选 X / Y 张，套餐精修 Z 张" |

#### Tab 3：修图协作

| 功能 | 描述 |
| --- | --- |
| 修图任务分配 | 下拉选择修图师，支持拆分为多人协作 |
| 修图进度 | 每张精修照片的状态：待修图 / 修图中 / 已上传 |
| 版本列表 | 每张精修照片的所有版本（v1, v2, v3...），可点击对比 |
| 版本对比 | 滑块对比模式：左侧原片、右侧精修版，拖动滑块切换 |
| 客户反馈 | 展示客户对每张照片的修改意见，状态：待处理 / 已处理 |

#### Tab 4：交付管理

| 功能 | 描述 |
| --- | --- |
| 交付链接 | 显示交付下载链接、有效期、已下载次数 |

### 3.4.3 交互规则

- Tab 之间切换不丢失已加载数据
- 上传原片后自动刷新原片列表
- 修图师上传精修版后，版本列表实时更新

# 4. 交互流程详细说明

## 4.1 核心流程：创建订单到客户选片

```
客服创建订单
    ↓
系统生成订单编号（ORD-YYYYMMDD-XXX）
    ↓
订单状态 → 待上传原片
    ↓
客服/摄影师进入订单详情 → 原片管理 Tab → 批量上传原片
    ↓
上传完成 → 系统自动生成缩略图
    ↓
订单状态 → 待选片
    ↓
系统生成选片链接：https://domain.com/s/{unique_token}
    ↓
客服点击"复制选片链接" → 发送给客户（微信/短信）
    ↓
客户打开链接 → 浏览照片 → 收藏 → 添加备注
    ↓
客户进入确认页 → 确认精修张数 → 提交选片
    ↓
订单状态 → 待修图
    ↓
系统通知修图师
```

## 4.2 核心流程：修图反馈收敛

```
修图师进入修图工作台 → 查看任务列表 → 进入任务详情
    ↓
查看客户选片结果和备注
    ↓
逐张精修 → 上传精修版（支持拖拽上传）
    ↓
全部上传完成 → 系统自动生成版本对比
    ↓
订单状态 → 待客户确认
    ↓
系统通知客户审阅精修（短信/H5推送）
    ↓
客户打开精修审阅页 → 逐张审阅
    ↓
┌─ 全部满意 → 点击"确认通过" → 订单状态 → 待交付
└─ 部分需修改 → 标注修改意见 → 提交 → 订单状态 → 待修改
    ↓
修图师收到修改通知 → 查看修改意见 → 上传新版本（v2, v3...）
    ↓
循环直到客户全部满意
```

## 4.3 状态流转规则

| 当前状态 | 触发条件 | 下一状态 |
| --- | --- | --- |
| 待上传原片 | 摄影师上传原片完成 | 待选片 |
| 待选片 | 客户提交选片确认 | 待修图 |

# 6. 业务规则

## 6.1 订单配额规则

| 规则 | 说明 |
| --- | --- |
| 免费版配额 | 每月最多创建 10 个订单，月末（自然月最后一天 23:59:59）重置计数 |
| 配额耗尽 | 提示"本月订单数已达上限，请升级至工作室版" |
| 升级后 | 保留所有历史订单，新订单不再受月度限制 |
| 取消订阅 | 当月剩余天数继续使用，次月恢复免费版限制 |

## 6.2 选片链接规则

| 规则 | 说明 |
| --- | --- |
| 链接唯一性 | 每个订单一个选片链接，token 为 64 位随机字符串 |
| 无需登录 | 客户打开链接无需注册/登录 |
| 密码保护 | 可选设置，开启后客户需输入密码或短信验证码 |
| 链接有效期 | 选片链接长期有效（直到订单完成） |
| 链接重生成 | 管理员可重新生成链接（旧链接失效） |

## 6.3 修图版本规则

| 规则 | 说明 |
| --- | --- |
| 版本编号 | 原片版本号为 0，第一次精修为 v1，依次递增 |
| 版本保留 | 所有版本永久保留，不可删除 |
| 版本对比 | 支持任意两个版本之间对比 |
| 反馈关联 | 每条反馈关联到具体照片+具体版本 |

## 6.4 交付链接规则

| 规则 | 说明 |
| --- | --- |
| 生成时机 | 客户确认全部满意后自动生成 |
| 默认有效期 | 30 天，可配置 7/15/30/60/90 天 |
| 下载次数 | 免费版不限次数；工作室版可设置次数限制 |
| 下载格式 | ZIP 打包下载（精修原图） |
| 防盗链 | 交付链接带签名，Referer 校验 |

## 6.5 通知规则

| 触发场景 | 通知对象 | 通知方式 |
| --- | --- | --- |
| 订单创建 | 摄影师 | 站内通知 |
| 原片上传完成 | 客服 | 站内通知 |
| 选片链接已发送 | - | - |
| 客户完成选片 | 修图师、客服 | 站内通知 + 短信（P1） |
| 精修版已上传 | 客户 | 短信通知（P1） |
| 客户提交修改意见 | 修图师 | 站内通知 |
| 客户确认满意 | 客服 | 站内通知 |
| 订单即将到期（提前3天） | 客服、管理员 | 站内通知 |

# 8. 非功能需求

## 8.1 性能指标

| 指标 | 目标值 |
| --- | --- |
| 单张原片上传（20MB，4G网络） | ≤ 10 秒 |
| 选片页首屏加载 | ≤ 3 秒 |
| 收藏/备注操作响应 | ≤ 500ms |
| 版本对比加载 | ≤ 2 秒 |
| 并发工作室数 | 50+ |
| 并发客户数 | 500+ |
| 缩略图生成 | ≤ 5 秒/张 |

## 8.2 安全要求

| 要求 | 说明 |
| --- | --- |
| 传输加密 | 全站 HTTPS（TLS 1.2+） |
| 数据隔离 | 不同工作室数据严格隔离，不可互访 |
| 链接安全 | 选片/交付链接 token 不可猜测，交付链接带签名 |
| 权限控制 | 基于角色的访问控制（RBAC） |
| 文件校验 | 上传文件校验类型和大小，防止恶意文件 |

## 8.3 兼容性

| 端 | 兼容性要求 |
| --- | --- |
| 工作室后台 | Chrome 90+, Edge 90+, Safari 15+，屏幕宽度 ≥1280px |
| 客户 H5 | 微信内置浏览器、Safari iOS 14+、Chrome Android 10+ |
| 屏幕适配 | H5 适配 375px-428px 宽度，兼顾平板和 PC |
