# AI模拟面试与话术优化助手V1.0 - 用户需求规格说明书

# 1.需求概述

## 1.1 需求介绍

AI模拟面试与话术优化助手是一款面向求职者的AI驱动面试练习工具，通过解析目标岗位JD（Job Description），模拟真实面试流程，按技术面/产品面/HR面等不同阶段进行提问，并对用户的文字或语音回答从多维度进行结构化评分，提供具体优化建议和参考话术，帮助求职者以低成本、高频次的方式系统性提升面试表现。

### 1.1.1 所属领域

在线教育科技、人力资源服务、AI应用

## 1.2 需求目标

- 为求职者提供基于真实岗位JD的AI模拟面试体验，覆盖技术面/产品面/HR面等多阶段面试场景
- 通过多维度结构化评分（流畅度、逻辑结构、关键信息覆盖、STAR法则等），帮助用户精准定位回答短板
- 提供针对性的优化建议和参考话术，让用户知道"怎么改"而不只是"哪里不好"
- 按岗位类型和面试阶段提供专项练习题库，支持用户有针对性地强化薄弱环节
- 记录每次练习的表现趋势，让用户直观看到自己的进步轨迹
- 以远低于人工面试教练（¥200-500/次）的成本，提供可高频使用的AI面试练习方案

## 1.3 系统使用角色

本系统主要服务于两类用户角色：
1. **求职者（C端用户）**：正在求职的应届生和职场人，准备跳槽的在职者，需要针对特定岗位练习面试的求职者
2. **平台运营方**：负责平台内容管理、题库维护、用户管理、订阅管理和数据运营的后台管理人员

## 时序图

```mermaid
sequenceDiagram
    actor User as 求职者
    participant App as 客户端
    participant Server as 后端服务
    participant LLM as AI大模型
    participant DB as 数据库

    Note over User, DB: 模拟面试完整流程

    User->>App: 上传岗位JD
    App->>Server: 提交JD文本/文件
    Server->>LLM: 解析JD，提取岗位信息
    LLM-->>Server: 返回结构化岗位信息
    Server->>App: 返回JD解析结果
    User->>App: 确认岗位信息，选择面试模式
    App->>Server: 创建面试会话
    Server->>LLM: 生成面试方案和首题
    LLM-->>Server: 返回面试方案+第一题
    Server->>App: 展示面试方案，开始提问

    loop 逐题问答
        User->>App: 文字/语音作答
        App->>Server: 提交用户回答
        par 并行处理
            Server->>LLM: 多维度评分
            Server->>LLM: 生成优化建议
            Server->>LLM: 生成参考话术
        end
        LLM-->>Server: 返回评分+建议+话术
        Server->>DB: 保存本轮评分记录
        Server->>App: 返回评分反馈
        App->>User: 展示评分和建议
        Server->>LLM: 生成下一题/追问
        LLM-->>Server: 返回下一题
        Server->>App: 展示下一题
    end

    User->>App: 结束面试
    Server->>LLM: 生成面试总结报告
    LLM-->>Server: 返回综合报告
    Server->>DB: 保存面试完整记录
    Server->>App: 返回面试报告
    App->>User: 展示完整报告
```
