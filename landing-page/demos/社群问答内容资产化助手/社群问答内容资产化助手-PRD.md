# 1. 产品概述

## 1.1 产品定位

社群问答内容资产化助手是一款面向知识创作者与社群运营人员的**轻量级内容后处理工具**。产品不替代飞书/企微等IM平台的社群管理能力，而是作为独立的"内容整理外挂"，帮助用户将付费社群、训练营、会员群中产生的大量群聊问答内容，通过AI自动化提取、整理、分类，输出为可复用的结构化内容资产（周报、FAQ、选题库、课程补充材料等），同时提供内容公开级别标记与自动脱敏功能，降低二次发布的合规风险。

## 1.2 产品价值主张

| 维度 | 价值描述 |
| --- | --- |
| 效率价值 | 将原本需要数小时的人工筛选整理工作缩短至分钟级 |
| 内容价值 | 将群内隐性知识转化为可检索、可发布、可变现的结构化内容资产 |
| 合规价值 | 通过分级标记+自动脱敏机制，降低二次发布的隐私与合规风险 |
| 商业价值 | 按量付费+订阅制双模式，降低试用门槛，提升付费转化 |

## 1.3 产品目标与成功指标

| 产品目标 | 成功指标（MVP阶段） | 目标值 |
| --- | --- | --- |
| 验证核心价值 | 用户完成一次完整"导入→提取→生成→导出"流程的比例 | ≥ 60% |
| 内容提取质量 | AI提取的问答对经用户审核后保留率 | ≥ 70% |
| 用户效率提升 | 单次整理任务耗时对比人工整理 | 缩短 ≥ 80% |
| 付费转化 | 免费试用用户转化为付费用户的比例 | ≥ 10% |

# 2. 用户画像与使用场景

## 2.1 核心用户画像

### 画像A：知识创作者 — "小李"

- **角色特征**：运营付费社群/训练营的个人IP，粉丝量5000-50000
- **核心痛点**：群内每天产生大量优质问答，但手动整理太耗时，很多好内容沉没在聊天记录中
- **使用目标**：每周从群聊中提取10-20条高质量问答，整理成公众号文章或课程补充材料
- **付费意愿**：愿意为省时间付费，月预算50-200元

### 画像B：社群运营人员 — "王姐"

- **角色特征**：某教育机构社群运营，同时管理5-10个社群
- **核心痛点**：每周需要产出各社群的周报给领导，手动翻阅聊天记录效率极低
- **使用目标**：快速汇总各群本周的精华问答，一键生成周报文档
- **付费意愿**：公司报销，对价格不太敏感，但对效率要求极高

### 画像C：课程助教 — "张老师"

- **角色特征**：在线课程平台的兼职助教，管理1-3个课程群
- **核心痛点**：学员反复问相同的问题，需要整理FAQ减少重复回答
- **使用目标**：从学员问答中提取高频问题和标准答案，构建课程FAQ文档
- **付费意愿**：个人付费，对价格敏感，倾向按量付费

## 2.2 核心使用场景

### 场景1：周报生成（高频场景）

**触发条件**：每周末，运营人员需要产出社群周报  
**操作流程**：上传本周群聊记录 → 系统自动提取问答 → 用户审核确认 → 选择"周报"模板 → 生成并导出Word文档  
**预期耗时**：10-15分钟（对比人工3-4小时）

### 场景2：FAQ文档整理（中频场景）

**触发条件**：课程助教发现学员频繁问相同问题  
**操作流程**：上传近期课程群聊天记录 → 系统提取问答并按话题聚类 → 用户筛选高价值问答 → 选择"FAQ"模板 → 生成FAQ文档  
**预期耗时**：15-20分钟

### 场景3：内容选题挖掘（中频场景）

**触发条件**：内容创作者寻找公众号/短视频选题灵感  
**操作流程**：上传多个社群的聊天记录 → 系统提取高价值问答 → 按话题分类浏览 → 选择"选题库"模板 → 导出选题列表  
**预期耗时**：10-15分钟

### 场景4：公开内容合规处理（贯穿场景）

**触发条件**：用户需要将群聊内容发布到公开平台  
**操作流程**：在内容审核环节标记公开级别 → 需匿名内容自动脱敏 → 预览脱敏效果 → 确认并导出  
**预期耗时**：嵌入在场景1-3中，额外增加2-3分钟

# 4. 功能详细设计

## 4.1 模块一：内容导入管理

### 4.1.1 文件上传

**功能目标**：让用户能快速将群聊记录文件上传至系统，并触发后续处理流程。

**功能规格**：

| 规格项 | 描述 |
| --- | --- |
| 支持格式 | txt（纯文本）、csv（表格）、json（结构化数据） |
| 兼容来源 | 微信、企业微信、飞书导出的聊天记录格式（MVP优先适配） |
| 单文件大小限制 | ≤ 50MB |
| 上传方式 | 点击上传区域选择文件 / 拖拽文件到上传区域 |
| 批量上传 | 支持同时选择多个文件，系统按文件名中的时间戳自动排序合并 |
| 上传反馈 | 展示上传进度条（百分比）；上传成功后显示文件摘要信息 |
| 错误处理 | 格式不支持时提示"不支持的文件格式，请上传txt/csv/json文件"；文件过大时提示"文件大小超过50MB限制" |

**交互流程**：

1. 用户进入"内容导入"页面
2. 选择目标社群（下拉选择或新建社群）
3. 点击上传区域或拖拽文件
4. 系统显示上传进度，完成后自动进行格式校验
5. 校验通过 → 显示文件预览（前20条消息）→ 用户确认
6. 校验失败 → 显示错误原因，引导用户重新上传

### 4.1.2 文件格式校验

**功能规格**：

| 校验项 | 规则 | 失败提示 |
| --- | --- | --- |
| 文件类型 | 必须是txt/csv/json之一 | "不支持的文件格式" |
| 文件大小 | ≤ 50MB | "文件大小超过50MB限制" |
| 文件编码 | UTF-8或GBK | "文件编码不支持，请使用UTF-8编码" |
| 内容格式 | 至少包含可识别的消息结构（时间戳+发送者+内容） | "未能识别聊天记录格式，请确认文件来源" |
| 空文件 | 文件内容不能为空 | "文件内容为空，请检查文件" |

### 4.1.3 文件预览

**功能规格**：

- 上传成功后展示前20条消息的预览卡片
- 每条消息显示：发送时间、发送者昵称、消息内容（截断至100字）
- 用户可确认"这是正确的聊天记录"或"重新上传"
- 预览区域顶部显示文件摘要：总消息条数、时间范围、识别的发送者数量

### 4.1.4 社群管理

**功能规格**：

| 功能点 | 描述 |
| --- | --- |
| 创建社群 | 填写社群名称（必填）、描述（选填）、来源平台（微信/企微/飞书/其他） |
| 编辑社群 | 修改社群基本信息 |
| 社群列表 | 卡片式展示：社群名称、来源平台图标、最近导入时间、内容总条数、未整理条数 |
| 删除社群 | 软删除，关联的导入记录和提取结果保留但标记为"已归档" |

### 4.1.5 导入记录

**功能规格**：

| 功能点 | 描述 |
| --- | --- |
| 记录列表 | 表格形式展示：导入时间、文件名、所属社群、消息条数、处理状态（处理中/已完成/失败） |
| 记录详情 | 点击查看该批次的处理详情：解析结果、提取结果、生成结果 |
| 删除记录 | 支持删除单条导入记录及其关联的原始数据（需二次确认） |
| 筛选 | 按社群、时间范围、处理状态筛选 |

## 4.2 模块二：智能内容提取

### 4.2.1 问答对识别

**功能目标**：从群聊记录中自动识别出有价值的"提问-回答"对话对。

**功能规格**：

| 规格项 | 描述 |
| --- | --- |
| 识别逻辑 | 基于LLM分析对话上下文，识别"问题+回答"的对话结构 |
| 过滤规则 | 自动过滤纯闲聊（如打招呼、表情互动）、无意义内容（如"收到"、"好的"） |
| 输出结构 | 每条识别结果包含：提问内容、回答内容、对话上下文（前后3条消息）、对话时间、参与者 |
| 案例识别 | 识别包含"问题描述+处理过程+解决方案"结构的完整案例 |
| 话题聚类 | 将相似问题自动归类，显示话题标签和该话题下的问答数量 |

### 4.2.2 价值评分

**功能规格**：

| 评分等级 | 判定标准 |
| --- | --- |
| 高价值 | 问题具有代表性（多人问过）或回答质量高（详尽、有实操指导意义） |
| 中价值 | 问题有一定参考价值，回答基本完整 |
| 低价值 | 问题过于个性化或回答较简单，参考价值有限 |

- 评分维度：问题代表性、回答质量、内容完整性、话题热度
- 评分结果显示在每条问答卡片的右上角，以颜色标签区分（高=绿色、中=橙色、低=灰色）

### 4.2.3 人工审核

**功能规格**：

| 功能点 | 描述 |
| --- | --- |
| 逐条审核 | 卡片式展示每条提取结果，用户可标记为"保留"、"删除"或"需修改" |
| 批量操作 | 全选/反选 → 批量保留、批量删除、批量标记公开级别 |
| 筛选排序 | 按价值评分、话题分类、时间范围筛选；按评分高低、时间排序 |
| 在线编辑 | 点击"需修改"进入编辑模式，可修正识别错误、补充上下文、调整文字内容 |
| 进度展示 | 顶部显示审核进度条：已审核X/Y条，保留Z条 |

### 4.2.4 公开级别标记

**功能规格**：

| 公开级别 | 含义 | 后续处理 |
| --- | --- | --- |
| 可公开 | 内容可直接对外发布，不涉及隐私信息 | 直接可用于输出导出 |
| 需匿名 | 内容有价值但包含敏感信息（人名、手机号等），需脱敏后使用 | 进入自动脱敏流程 |
| 不可外发 | 内容仅限系统内查看，不可对外发布 | 禁止导出，仅可在线查看 |

- 默认标记规则：系统根据内容是否包含敏感信息给出初始建议标记，用户可修改
- 批量标记：按话题或价值等级批量设置公开级别
- 标记结果以颜色标签显示在卡片上（可公开=绿色、需匿名=橙色、不可外发=红色）

### 4.2.5 自动脱敏

**功能规格**：

| 规格项 | 描述 |
| --- | --- |
| 触发条件 | 用户将内容标记为"需匿名"后自动触发 |
| 脱敏范围 | 人名/昵称、手机号、邮箱、身份证号、银行卡号、详细地址 |
| 脱敏方式 | 替换为占位符，如"张三"→"[姓名A]"、"13800138000"→"[手机号A]" |
| 同一性保持 | 同一人在不同问答中被替换为相同的占位符（如[姓名A]） |
| 脱敏预览 | 展示脱敏前后的内容对比，用户可确认或手动调整脱敏结果 |
| 确认脱敏 | 用户确认脱敏效果后，内容状态变更为"已脱敏"，可用于后续输出 |

## 4.3 模块三：内容整理与输出

### 4.3.1 输出模板

**MVP预设模板**：

| 模板名称 | 适用场景 | 内容结构 |
| --- | --- | --- |
| 社群周报 | 每周社群运营汇报 | 本周概览 → 精华问答TOP5 → 话题分类统计 → 下周待关注话题 |
| FAQ文档 | 整理高频问答为标准FAQ | 分类目录 → 各FAQ条目（问题+标准答案+来源标签） |
| 选题库 | 内容创作者选题参考 | 选题列表（话题+核心问题+热度评分+建议角度） |
| 课程补充材料 | 补充课程内容 | 知识模块分类 → 补充问答集 → 学员常见误区汇总 |
| 知识卡片 | 单条知识内容卡片 | 标题 → 核心问答 → 关键词标签 → 适用场景 |

- 模板预览：选择模板后可查看该模板的输出样式和结构说明
- V2.0计划：支持自定义模板编辑、品牌模板（Logo/颜色/页眉页脚）

### 4.3.2 内容生成

**功能规格**：

| 功能点 | 描述 |
| --- | --- |
| 内容选择 | 用户从已审核的问答中选择要整理的内容（支持全选、按话题选择、按评分选择） |
| 模板选择 | 选择输出模板，可预览模板结构 |
| 一键生成 | 点击"生成"后系统调用LLM按模板结构整理内容，显示生成进度 |
| 生成预览 | 以富文本形式展示生成结果，各板块可折叠展开 |
| 在线编辑 | 用户可在预览页面直接编辑内容：修改文字、调整段落顺序、增删板块 |
| 重新生成 | 对单个板块可点击"重新生成"让LLM重新整理该部分内容 |

### 4.3.3 导出管理

**功能规格**：

| 规格项 | 描述 |
| --- | --- |
| 支持格式 | Markdown(.md)、Word(.docx)、PDF(.pdf) |
| 导出设置 | 选择格式后可配置：是否包含目录、是否显示来源社群名称、日期范围 |
| 导出反馈 | 导出完成后自动触发下载，同时显示"查看导出历史"入口 |
| 导出历史 | 列表展示所有历史导出文件：导出时间、模板类型、文件格式、文件大小；支持重新下载 |

### 4.3.4 周报自动化（订阅版功能）

**功能规格**：

| AI提取结果 | 数据库 | 随用户账户保留 |
| 导出文件 | 对象存储(OSS) | 保留90天，到期后提示重新下载 |

# 7. 业务规则

## 7.1 计费规则

| 规则编号 | 规则描述 |
| --- | --- |
| BR-01 | 字数计算以导入的原始文件字符数为准，不以AI提取后的字符数计算 |
| BR-02 | 不足1万字按1万字计算，例如导入1.2万字文件消耗2万字额度 |
| BR-03 | 免费额度5000字在注册时一次性发放，不可恢复 |
| BR-04 | 按量充值的额度无有效期限制 |
| BR-05 | 月度订阅到期后，已有内容可继续查看但不可新建导入/提取任务 |
| BR-06 | 导出操作不消耗字数额度 |
| BR-07 | 重新处理（如对同一文件重新提取）不重复消耗字数 |

## 7.2 内容安全规则

| 规则编号 | 规则描述 |
| --- | --- |
| BR-10 | 用户上传的聊天内容仅在用户账户内可见，不对其他用户公开 |
| BR-11 | 标记为"不可外发"的内容禁止通过任何导出功能输出 |
| BR-12 | 脱敏处理为辅助工具，系统提示用户自行审核脱敏结果 |
| BR-13 | 用户数据不用于模型训练 |

## 7.3 处理状态规则

| 导入状态 | 含义 | 允许操作 |
| --- | --- | --- |
| 待处理 | 文件已上传，等待处理 | 查看文件预览、取消处理 |
| 处理中 | 正在进行AI提取 | 查看进度、取消处理 |
| 已完成 | 提取完成，等待审核 | 审核、标记、生成、导出 |
| 处理失败 | 处理过程出错 | 重新处理、删除 |
