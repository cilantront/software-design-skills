# 1. 产品定位

## 1.1 产品愿景

为高频预约制服务门店（口腔诊所、医美/康复门店等）提供一套**从"被动盘点"到"主动预测"**的耗材智能管理工具，通过预约数据驱动耗材缺口预测，让门店不再因耗材过期而浪费、不再因临时缺货而停诊。

## 1.2 目标用户

| 角色 | 用户画像 | 核心诉求 |
|------|----------|----------|
| 门店操作员（护士/前台） | 每天接待 20+ 患者的口腔诊所护士，需要快速确认手套、麻药是否够用 | 快速入库、即时知道耗材够不够用 |
| 门店店长/负责人 | 管理 3–5 名护士的医美门店店长，需控制耗材成本并保证不断供 | 审核补货清单、控制耗材成本 |
| 连锁运营负责人 | 管理 10–20 家门店的连锁运营总监，需一眼看到哪些门店耗材告急 | 多门店全局监控、统一采购策略 |

## 1.3 产品价值主张

| 价值点 | 量化目标 |
|--------|----------|
| 降低耗材过期浪费 | 临期耗材浪费率降低 50% 以上 |
| 避免缺货断供 | 缺货事件减少 80% 以上 |
| 减少人工盘点成本 | 盘点时间从每天 30 分钟降至 5 分钟 |
| 支撑规模化运营 | 单人管理门店数从 5 家提升至 15 家 |

## 1.4 商业模式

| 版本 | 定价 | 目标客户 |
|------|------|----------|
| 单店版 | ¥99/月 | 独立口腔诊所、医美门店 |
| 连锁版 | ¥99 × 门店数 × 0.8（阶梯折扣） | 3 家以上连锁门店 |

## 1.5 MVP 范围

**MVP 周期**：1–2 周  
**MVP 交付功能**（对应 P0 优先级）：
- 耗材手动入库
- 库存台账（含批次管理）
- 安全库存设置
- 预约数据手动录入与汇总
- 消耗规则配置
- 7 天耗材缺口预测
- 补货清单生成与导出（Excel）
- 临期批次预警与清单
- 低库存 / 临期消息提醒

# 2. 功能详述

## 2.1 耗材入库管理

### 2.1.1 手动录入入库（P0）

**功能说明**：用户通过 WEB 端表单手动录入耗材入库信息。

**输入字段**：

| 字段 | 类型 | 必填 | 规则 |
|------|------|------|------|
| 耗材名称 | 文本 | 是 | 最大 50 字符 |
| 规格型号 | 文本 | 是 | 最大 100 字符 |
| 批次号 | 文本 | 是 | 同一耗材+批次号不可重复入库（需合并数量） |
| 生产日期 | 日期 | 是 | 不得晚于当天 |
| 有效期 | 日期 | 是 | 必须晚于生产日期 |
| 入库数量 | 整数 | 是 | 大于 0 |
| 单位 | 下拉选择 | 是 | 盒/支/个/包/瓶/套 |
| 供应商 | 文本 | 否 | 最大 100 字符 |

**业务规则**：
- 同一耗材+同一批次号再次入库时，系统自动合并数量，不创建新记录
- 入库后库存台账实时更新
- 入库时若有效期距今天 ≤ 预警天数（默认 30 天），立即触发临期提醒
- 入库后若该耗材总库存 < 安全库存，仍保持低库存预警状态

**验收标准**：
- 单品录入操作在 30 秒内完成
- 入库成功后库存台账数量即时更新
- 临期批次入库时页面弹出临期提醒 Toast
- 重复批次号入库时数量自动合并

### 2.1.2 入库记录查询（P1）

**功能说明**：查看历史入库记录，支持多维度筛选。

**筛选条件**：日期范围、耗材名称（模糊搜索）、批次号。  
**列表字段**：入库时间、耗材名称、规格、批次号、有效期、数量、操作人。  
**操作**：支持更正（需记录更正原因和更正前后值）。

## 2.2 库存台账管理

### 2.2.1 库存总览（P0）

**功能说明**：以列表形式展示所有耗材的当前库存状态。

**列表字段**：

| 字段 | 说明 |
|------|------|
| 耗材名称 | 耗材通用名 |
| 规格 | 规格型号 |
| 当前库存 | 所有批次可用数量之和 |
| 安全库存 | 用户设定的安全库存阈值 |
| 库存状态 | 正常（绿）/ 低库存（黄）/ 缺货（红） |
| 最近批次有效期 | 所有批次中最短的有效期 |

**筛选与排序**：支持按名称搜索、按状态筛选、按库存量排序。

### 2.2.2 批次明细查看（P0）

**功能说明**：点击某耗材可查看其所有在库批次的明细。

**展示信息**：批次号、入库日期、有效期、剩余数量、状态（正常/临期/过期）。  
**操作**：支持对该批次进行出库（使用）或报损操作。

### 2.2.3 出库/报损登记（P0）

**功能说明**：记录耗材出库（临床使用）或报损操作。

**输入字段**：操作类型（出库/报损）、耗材名称、批次号、操作数量、操作原因。  
**业务规则**：
- 出库后库存实时扣减
- 报损后库存扣减并记录报损原因
- 出库优先扣减有效期最近的批次（FEFO — First Expired First Out）
- 操作不可逆，需二次确认

## 2.3 安全库存管理

### 2.3.1 手动设置阈值（P0）

**功能说明**：为每种耗材设置安全库存下限值。

**交互方式**：在库存台账页面直接编辑安全库存列，或在独立的"安全库存设置"页面批量管理。  
**业务规则**：
- 安全库存值必须 ≥ 0
- 修改后即时生效，系统重新评估所有耗材的库存状态
- 安全库存为 0 表示不启用该耗材的低库存预警

### 2.3.2 智能推荐阈值（P1）

**功能说明**：系统根据近 30 天平均日消耗量 × 补货周期天数（默认 7 天）自动推荐安全库存值。  
**交互方式**：在设置页面显示推荐值，用户可一键采纳或手动调整。

## 2.4 预约数据管理

### 2.4.1 手动录入预约（P0）

**功能说明**：用户手动录入门店预约信息。

**输入字段**：

| 字段 | 类型 | 必填 | 规则 |
|------|------|------|------|
| 预约日期 | 日期 | 是 | 今天起 30 天内 |
| 患者姓名 | 文本 | 否 | 最大 20 字符（可选，保护隐私） |
| 预约项目 | 下拉选择 | 是 | 从已配置的项目列表中选择 |
| 预约数量 | 整数 | 是 | 默认 1，表示该项目的预约例数 |
| 备注 | 文本 | 否 | 最大 200 字符 |

### 2.4.2 预约量汇总（P0）

**功能说明**：自动汇总未来 7 天各项目的预约数量。

**展示方式**：按日期分列，按项目汇总，显示每个项目每天的预约例数和 7 天合计。

### 2.4.3 消耗规则配置（P0）

**功能说明**：配置每个服务项目对应的耗材消耗种类和数量。

**配置示例**：

| 服务项目 | 耗材名称 | 单次消耗量 |
|----------|----------|------------|
| 洗牙 | 一次性手套 | 1 副 |
| 洗牙 | 口腔检查镜 | 1 个 |
| 玻尿酸注射 | 玻尿酸 | 1 支 |
| 玻尿酸注射 | 一次性注射器 | 1 支 |
| 水光针 | 水光针套装 | 1 套 |

**业务规则**：
- 一个项目可关联多种耗材
- 消耗量支持小数（如 0.5 盒）
- 修改消耗规则后，下次预测计算时生效

## 2.5 耗材缺口预测

### 2.5.1 7 天缺口预测计算（P0）

**功能说明**：系统根据预约量与消耗规则自动计算各耗材未来 7 天的预计消耗量和缺口。

**计算公式**：
```
预计消耗量 = Σ (未来7天每天的预约例数 × 该项目对该耗材的单次消耗量)
缺口量 = 预计消耗量 - 当前可用库存
```

**展示字段**：

| 字段 | 说明 |
|------|------|
| 耗材名称 | 耗材通用名 |
| 当前库存 | 当前可用数量 |
| 预计消耗量 | 未来 7 天预计消耗总量 |
| 缺口量 | 正数=不足需补货，0 或负数=库存充足 |
| 库存状态 | 充足（绿）/ 需补货（红） |

**业务规则**：
- 预测计算在用户触发时执行（点击"刷新预测"按钮）
- 计算结果实时展示，无需等待
- 缺口项用红色高亮标注并排在列表顶部

### 2.5.2 缺口可视化图表（P1）

**功能说明**：以柱状图对比展示各耗材的当前库存 vs 预计消耗量，直观呈现缺口。

## 2.6 补货清单管理

### 2.6.1 补货清单自动生成（P0）

**功能说明**：根据缺口预测结果自动生成补货清单。

**清单字段**：

| 字段 | 说明 |
|------|------|
| 耗材名称 | 需补货的耗材 |
| 规格 | 规格型号 |
| 缺口数量 | 预测缺口量 |
| 建议采购量 | 缺口量向上取整至最小包装单位 |
| 紧急程度 | 紧急（预计 2 天内断货）/ 一般（3-7 天内断货） |
| 建议到货日期 | 预计断货日期前一天 |

### 2.6.2 补货清单导出（P0）

**功能说明**：将补货清单导出为 Excel 文件。

**导出格式**：.xlsx 文件，包含清单全部字段和生成日期。  
**文件命名规则**：`补货清单_{门店名称}_{YYYYMMDD}.xlsx`

### 2.6.3 清单审核（P1）

**功能说明**：店长可审核补货清单，调整采购数量后确认。  
**业务规则**：确认后的清单标记为"已确认"状态，可后续查询。

## 2.7 临期耗材管理

### 2.7.1 临期批次预警（P0）

**功能说明**：系统每日自动扫描所有在库批次，识别距有效期达到预警天数的临期批次。

**预警规则**：
- 默认预警阈值：到期前 30 天
- 用户可自定义预警天数（30 / 60 / 90 天）
- 预警扫描频率：每日凌晨 02:00 自动执行

### 2.7.2 临期耗材清单（P0）

**功能说明**：展示所有临期耗材的详细信息。

**列表字段**：

| 字段 | 说明 |
|------|------|
| 耗材名称 | 耗材通用名 |
| 批次号 | 临期批次 |
| 有效期 | 到期日期 |
| 剩余天数 | 距到期日的天数 |
| 剩余数量 | 该批次当前库存量 |
| 状态 | 临期（黄）/ 即将过期（红，≤7 天） |

**排序规则**：按剩余天数升序排列（最先过期的排在最前）。

### 2.7.3 临期消耗匹配分析（P1）

**功能说明**：将临期库存量与未来 7 天预约预计消耗量对比。

**展示逻辑**：
- 若预约消耗量 ≥ 临期库存量：标记"可消耗完"，建议优先使用该批次
- 若预约消耗量 < 临期库存量：标记"无法完全消耗"，提示需调整预约排期或准备报损

## 2.8 消息提醒中心

### 2.8.1 低库存提醒（P0）

**触发条件**：某耗材当前库存 < 安全库存阈值。  
**推送对象**：门店操作员 + 门店店长。  
**提醒内容**：`[低库存预警] {耗材名称} 当前库存 {N}，已低于安全库存 {M}，请及时补货。`  
**推送方式**：站内信（MVP），后续迭代支持微信/短信。

### 2.8.2 临期提醒（P0）

**触发条件**：某耗材批次进入临期预警范围。  
**推送对象**：门店操作员 + 门店店长。  
**提醒内容**：`[临期预警] {耗材名称}（批次 {BATCH}）将于 {DATE} 到期，剩余 {N} {单位}，请优先安排使用。`  
**推送频率**：每天汇总推送一次（避免打扰）。

### 2.8.3 提醒消息列表（P0）

**功能说明**：在"提醒页"展示所有未读和已读提醒消息。

**消息分类**：低库存预警、临期预警、系统通知。  
**操作**：标记已读、全部已读、删除。

```
预约（Appointment）
├── id, date, patient_name
├── service_item_id, quantity
└── created_at

服务项目（ServiceItem）
├── id, name, category
└── consumption_rules[] → 关联耗材与消耗量

消耗规则（ConsumptionRule）
├── id, service_item_id, material_id
└── unit_consumption (单次消耗量)

补货清单（RestockList）
├── id, store_id, generated_at
├── status (草稿/已确认)
├── items[] → 补货明细
└── operator_id

提醒消息（Notification）
├── id, type (低库存/临期/系统)
├── target_user_ids[]
├── content, is_read
└── created_at
```

# 4. 验收标准

## 4.1 MVP 验收场景

### 场景 A：口腔诊所日常耗材入库

| 验收项 | 通过标准 |
|--------|----------|
| 手动录入 | 护士在入库页面填写耗材名称、规格、批次号、生产日期、有效期、数量，提交后 30 秒内完成 |
| 库存更新 | 入库成功后库存台账数量即时更新 |
| 临期提醒 | 入库时若有效期 ≤ 30 天，页面弹出临期提醒 |
| 重复批次 | 同批次号再次入库时数量自动合并 |

### 场景 B：医美门店预约预测与补货

| 验收项 | 通过标准 |
|--------|----------|
| 预约录入 | 店长可录入未来 7 天各项目的预约例数 |
| 消耗规则 | 每个项目已配置对应耗材和消耗量 |
| 缺口预测 | 点击"刷新预测"后，5 秒内展示各耗材的当前库存、预计消耗量和缺口 |
| 缺口高亮 | 缺口项（正数）用红色高亮并排在顶部 |
| 补货清单 | 一键生成补货清单，包含缺口数量和建议采购量 |
| 清单导出 | 导出为 .xlsx 文件，文件命名正确，内容完整 |

### 场景 C：临期耗材管理

| 验收项 | 通过标准 |

# 5. 页面结构与交互说明

## 5.1 导航结构

```
侧边栏导航
├── 库存管理（库存总览 + 批次明细）
├── 耗材入库
├── 预约录入
├── 预测补货（缺口预测 + 补货清单）
├── 临期管理
├── 提醒中心
└── 系统设置（安全库存设置 + 消耗规则配置 + 预警阈值配置）
```

## 5.2 页面交互要点

### 库存管理页
- 列表支持搜索、筛选、排序
- 点击耗材名称展开批次明细（行内展开或弹窗）
- 库存状态列使用颜色圆点标识（绿/黄/红）
- 安全库存列支持行内编辑

### 预约录入页
- 支持按日期切换查看（日历视图或列表视图）
- 录入后实时显示当日预约汇总
- 项目下拉列表来自系统配置的消耗规则

### 预测补货页
- 顶部显示预测概览（需补货耗材数量、最紧急断货日期）
- 中部为缺口明细表格
- 底部为补货清单操作区（生成清单、导出 Excel）

### 提醒页
- 顶部为消息分类标签（全部 / 低库存 / 临期 / 系统）
- 消息列表按时间倒序
- 未读消息有加粗标识和蓝色圆点
- 支持"全部标记已读"

# 附录

## A. 术语表

| 术语 | 定义 |
|------|------|
| 耗材 | 医疗服务过程中使用的一次性物品或消耗性材料 |
| 批次 | 同一生产日期、同一有效期、同一供应商供货的耗材 |
| 安全库存 | 系统预设的库存下限阈值 |
| 临期耗材 | 距有效期达到预警天数的耗材批次（默认 30 天） |
| 缺口 | 预测消耗量超出当前可用库存的数量差值 |
| 消耗规则 | 每个服务项目对应的耗材消耗种类和数量的映射关系 |
| FEFO | First Expired First Out，先到期先出库原则 |
| UDI | 医疗器械唯一标识（Unique Device Identification） |
