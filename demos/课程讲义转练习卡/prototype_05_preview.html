#1 选择题 中等
以下哪项不属于项目管理的五大过程组？
A 启动
B 规划
C 执行
D 运营 ✓
正确答案：D
项目管理的五大过程组包括：启动、规划、执行、监控、收尾。运营属于日常持续性活动，不属于项目管理过程组。项目是临时性的工作，而运营是持续重复的活动。
📌 关联知识点：项目管理五大过程组
#2 选择题 中等
项目管理的"铁三角"约束不包括以下哪项？
A 范围
B 质量 ✓
C 时间
D 成本
正确答案：B
项目管理的铁三角（三重约束）包括：范围、时间、成本。质量虽然重要，但不属于传统的铁三角约束，而是受三者影响的结果。
📌 关联知识点：项目管理的价值
#3 选择题 中等
以下哪个文档正式授权项目的存在？
A 项目章程 ✓
B 项目管理计划
C 范围说明书
D 商业论证
正确答案：A
项目章程是正式授权项目存在的文件，由项目发起人签发。它赋予项目经理使用组织资源的权力。
📌 关联知识点：项目的定义与特征
#4 判断题 中等
项目的临时性意味着项目的持续时间一定很短。
正确答案：错
项目的临时性是指每个项目都有明确的起点和终点，但并不意味着持续时间短。一个项目可能持续数月甚至数年。临时性强调的是"有终点"而非"短暂"。
📌 关联知识点：项目的生命周期
#5 判断题 中等
项目与日常运营的共同特征是都具有重复性。
正确答案：错
项目具有独特性，每个项目的产出都是独特的；而日常运营才具有重复性。项目与运营的本质区别在于：项目是临时的、独特的，运营是持续的、重复的。
📌 关联知识点：项目的定义与特征
