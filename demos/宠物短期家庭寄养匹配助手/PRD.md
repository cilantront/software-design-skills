# 1 概述

## 1.1 需求背景

中国宠物数量已超 1 亿只，宠物主人出差/旅行时的寄养需求巨大且高频。传统宠物店寄养普遍存在以下问题：

**核心痛点：**
- **价格高昂**：宠物店寄养 80-200 元/天，且多数采用笼养方式，空间局促
- **环境不透明**：主人无法实时了解宠物在门店的生活状态，焦虑感强
- **应激风险**：笼养环境下宠物易产生应激反应，尤其对性格敏感的猫/狗
- **供给真空**：早期宠物寄养平台（如"小狗在家"）已停运或萎缩，美团/58 等平台信息杂乱且无保障

**业务价值：**
- 聚焦"短期家庭寄养"细分场景，提供比宠物店更温馨、性价比更高（30-80 元/天）的寄养选择
- 通过平台认证审核、寄养日志、宠物意外险等机制构建信任闭环
- 为有养宠经验的个人提供共享经济下的服务变现渠道

**预期目标：**
- MVP 10-14 天上线核心链路：寄养家庭入驻认证 → 搜索匹配 → 预约支付 → 寄养日志 → 评价结算
- 首版支持 1000 并发用户在线，覆盖宠物主人端小程序 + 寄养家庭端小程序 + 平台管理后台 WEB 三端
- 平台每笔订单抽佣 15%，附加宠物意外险（¥10-30/次）与推广置顶（¥20/月）增值服务

## 1.3 产品介绍

宠物短期家庭寄养匹配助手是一款面向经常出差/旅行宠物主人的服务平台，连接经过认证的寄养家庭与有短期寄养需求的宠物主人，差异化于传统宠物店笼养模式，让宠物在主人外出期间获得家庭式的悉心照料。

**目标用户：**
- 经常出差/旅行的宠物主人（猫/狗为主）
- 有养宠经验且愿意提供家庭寄养服务的个人
- 对宠物店笼养不放心、希望给宠物找"家庭式"寄养环境的主人

**核心使用场景：**
1. 宠物主人出差/旅行前，通过平台按位置、宠物类型、价格筛选附近寄养家庭，查看评价后在线预约并支付
2. 寄养期间，寄养家庭每日发送宠物状态日志（照片/短视频），宠物主人随时查看并通过平台在线沟通
3. 寄养到期后，宠物主人接宠并对寄养家庭进行评价，平台结算费用至寄养家庭（扣除 15% 服务费）

**产品核心价值：**
- **安心匹配**：认证审核 + 评价浏览 + 环境实拍，选得放心
- **透明寄养**：每日日志 + 在线沟通，随时了解宠物近况
- **高性价比**：家庭寄养 30-80 元/天，比宠物店节省 50%+
- **安全保障**：宠物意外险覆盖寄养期间，平台协调纠纷

| 项 | 内容 |
| --- | --- |

## 3.1 宠物主人端（小程序端）功能

### 3.1.1 微信授权登录
功能描述：支持用户通过微信小程序授权快速登录，一键获取微信头像、昵称等信息完成注册/登录。同时支持手机号+验证码注册登录作为备选方式。

*优先级与依赖说明：*
| 项 | 内容 |
| --- | --- |
| 优先级 | P0 |
| 依赖需求 | 无 |
| 前置条件 | 用户已安装微信客户端（版本≥7.0） |

### 3.1.2 微信授权登录—详细流程

```mermaid
flowchart TD

    B --> C{用户点击允许授权?}

    C -->|否| E[展示手机号登录备选方案]
```

2. 首次登录自动创建用户，引导完善宠物档案
3. 登录 token 有效期 7 天，过期自动刷新

## 7.2 后续迭代规划

- V1.1（上线后1个月）：增加地图模式搜索、宠物主人端增加寄养家庭收藏功能、寄养家庭端增加档期批量设置
- V1.2（上线后2个月）：增加宠物主人端"常去寄养家庭"快捷下单、寄养家庭端增加多宠物同时寄养优惠设置
- V1.3（上线后3个月）：增加宠物健康档案（疫苗记录、过敏史）、平台管理后台增加智能推荐算法（根据宠物特征推荐寄养家庭）
- V2.0（上线后6个月）：增加宠物主人端"寄养社区"（分享寄养经验）、增加宠物电商入口（寄养家庭推荐宠物用品）
